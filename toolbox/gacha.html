🎰 ガチャ確率計算

「当たる確率 / 引く回数 / 当選確率(少なくとも1回当たる確率)」を相互に計算。金額も出せます。

当たる確率（1回あたり） p
入力例：0.006 / 0.6% / 1/166
引く回数 n
整数で入力（例：100）
当選確率（少なくとも1回当たる確率） q
入力例：0.5 / 50%
1回あたりの金額（任意）
未入力なら金額計算はスキップ
10連の金額（任意）
多くのガチャの「10連で少し安い」を反映
使い方
・p と n を入れる → q（少なくとも1回当たる確率）を計算
・p と q を入れる → 必要な n（回数）を計算
・n と q を入れる → 必要な p（1回あたり確率）を計算
・金額：1回の金額（任意）と 10連の金額（任意）を入れると、回数に対する総額を表示
